# soal2
Program ini menghitung faktorial dari bilangan bulat positif.
Jika angka yang dimasukkan negatif, muncul pesan error.
Kalau positif, program menghitung hasil perkalian dari 1 sampai n dan menampilkan hasil faktorialnya.
